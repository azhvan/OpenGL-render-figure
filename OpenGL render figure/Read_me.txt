Инструкция
		

		Описание осей координат:
		Ось x нарисована красным цветом
		Ось y нарисована зеленым цветом
		Ось z нарисована синим цветом


		Смена фигуры:
		Для смены отображения вектора и курсора используйте клавишу \"Tab\"

		
		Функции с фугурами:

		Функция 1:
		Для поворота объекта нужно сначала выбрать функцию под номером 1. Для ее выбора нажмите 1.
		Для смены направления вращения используйте клавиши \"+\" и \"-\".
		Для произведения вращения нажмите на клавишу которая отвечает за букву оси вокруг которой будет вращение. (x, y, z)

		Функция 2:
		Для передвижения объекта нужно сначала выбрать функцию под номером 2. Для ее выбора нажмите 2.
		Для смены направления движения используйте клавиши \"+\" и \"-\".
		Для произведения движения нажмите на клавишу которая отвечает за букву оси по которой будет движение. (x, y, z)

		Функция 3:
		Для изменения размеров объекта нужно сначала выбрать фунцию под номером 3. Для ее выбора нажмите 3.
		Для произведения смены движения используйте кливиши \"+\" и \"-\".

		Смена цвета:
		Для смены цвета объекта на красный нажмите на клавишу \"r\"
		Для смены цвета объекта на синий нажмите на клавишу \"b\"
		Для смены цвета объекта на зеленый нажмите на клавишу \"g\"
		Для смены цвета объекта на белый нажмите на клавишу \"w\"

		Вращение камеры:
		Для вращения камеры используйте стрелки вверх, вниз, вправо, влево. Так же есть возможнлсть управления камерой с помощью мыши.
		Для этого зажмите левую кнопку мышь и двигайте мышь в нужную вам сторону.
